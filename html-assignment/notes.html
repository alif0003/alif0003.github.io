HTML Tags I learned

Headings
We can use the <h1> tag for the most important heading of the page. For the next level of heading we have <h2>, <h3>, <h4>, <h5>, and <h6>. The browser will style each heading differently, stating with the biggest size for <h1> then <h2> and so on.
Body
Is the main body of the document, this is what we see in our browsers. All the content of the web page must be added inside the opening and closing tags <body>.
Emphasis
A string of emphasized, slightly more important text. Screen readers will change their voice for this text <em>.
Highlighting text
Used to highlight a piece of text for reference. The keywords in a search results page, the current navigation item, etc <mark>.
Subscript text
Defines text as being subscript <sub> H2O
